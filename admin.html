🔗 Cloudflare Worker — Auto-remplissage Affiliés
⚡ Ce Worker permet de récupérer automatiquement le titre, l'image, la description et le prix depuis n'importe quel lien affilié (Amazon, Etsy, AliExpress…).
Déployez le fichier og-worker.js sur Cloudflare Workers (gratuit · 100 000 req/jour), puis collez l'URL ici.
📋 Instructions de déploiement :
1. Allez sur dash.cloudflare.com → Workers & Pages → Create Worker
2. Collez le code du fichier og-worker.js → Save and Deploy
3. Copiez l'URL générée (ex: og-extractor.xxx.workers.dev) → collez-la ci-dessous
🔗 Synchronisation GitHub — Publier les modifications
⚡ Entrez votre token GitHub une seule fois. Ensuite, chaque modification (produits, images, blog) sera publiée automatiquement sur votre site pour tous les visiteurs.
